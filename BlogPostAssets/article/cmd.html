cmd命令大全

实用工具

感谢@GSC大佬提供命令 —— orz

  • clearmgr : 清理垃圾
  • taskmgr : 任务管理器
  • eventvwr : 事件管理器
  • shutdown -s -t 10 : 关机
  • shutdown -r -t 10 : 重启
  • osk : 屏幕键盘
  • cls : 清屏
  • calc : 计算器
  • exit : 关闭
  • explorer "http://www.luogu.org/login/loginpage" : 网页
  • copy 1.txt /b + 2.jpg/a 3.jpg : 将1.txt中的内容加到2图片中并生成一个新的3.jpg
  • cd : 加入目录
  • 360se.exe : 打开360浏览器(在目录中直接写文件名即为打开此文件)
  • e: : 切换到E盘
  • dir : 显示目录中所有文件或目录
  • notepad : 记事本
  • compmgmt.msc : 计算机管理
  • dvdplay : DVD播放器
  • charmap : 字符映射表
  • chskdsk.exe : 磁盘检查
  • progman : 程序管理器
  • perfmon.msc : 计算机性能检测
  • wupdmgr windows : 程序更新
  • write : 写字板
  • mspaint : 画图板
  • mstsc : 远程桌面连接
  • magnify : 放大镜实用程序
  • eudcedit : 造字程序
  • fc 1.txt 2.txt > 3.txt : 对比1.txt和2.txt并将不同输出至3.txt
  • findstr "Hello" a.txt : 在a.txt中找字串"Hello", 直接在屏幕上输出含有Hello那一行的信息
  • md abc : 在本目录下创建一个叫abc的文件夹
  • ren 原文件名 新文件名 : 给文件改名
  • type 文件名 : 显示文件内容
  • copy 1.txt C:\users\****\Desktop : 将本目录下的1.txt复制粘贴到C:\users\****\Desktop目录下
  • copy *,* D:temp : 将当前目录下所有文件复制粘贴到D:\temp下
  • del /s /q 目录 : 删除目录及目录下所有子目录及文件, 参数\q可跳过系统确认 (黑客操作)
  • a.exe < testdata.txt > 1.txt : 将\(testdata.txt\)中的数据输入\(a.exe\)并将结果输出到\(1.txt\)
